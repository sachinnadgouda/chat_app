# CHAT
This is a real time chat application based on MEAN stack.

# Instructions to run
Clone the project
git clone git@github.com:sachinnadgouda/chat_app.git


DataBase - Mongo
Check if mongodb service is running in your machine else start the service.

Server
Node and npm are installed in your machine.
Install all dependencies using  
   `npm install`  
   `node server.js`  
  
Your server will be setup and ready for use.

UI
Open browser and hit `localhost:8080`.
Register user.
Login from the same screen.
